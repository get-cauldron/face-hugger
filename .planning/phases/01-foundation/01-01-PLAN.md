**`src-tauri/src/state.rs`:**
```rust
use serde::{Deserialize, Serialize};
use tokio::sync::Mutex;

#[derive(Debug, Clone, Serialize, Deserialize, specta::Type)]
pub struct UserInfo {
    pub name: String,
    #[serde(rename = "fullname")]
    pub fullname: Option<String>,
    #[serde(rename = "avatarUrl")]
    pub avatar_url: Option<String>,
    pub email: Option<String>,
    #[serde(rename = "type")]
    pub user_type: Option<String>,
}

#[derive(Debug, Default)]
pub struct AuthState {
    pub token: Option<String>,
    pub user: Option<UserInfo>,
}

pub struct AppState {
    pub auth: Mutex<AuthState>,
}

impl Default for AppState {
    fn default() -> Self {
        Self {
            auth: Mutex::new(AuthState::default()),
        }
    }
}
```

**`src-tauri/src/hf/client.rs`:**
Create a function `pub async fn whoami(token: &str) -> Result<UserInfo, String>` that:
- Creates a `reqwest::Client`
- Calls `GET https://huggingface.co/api/whoami-v2` with `Authorization: Bearer {token}` header
- On non-success status: returns `Err("Invalid token — check your HF settings".to_string())`
- On success: deserializes response JSON into `UserInfo` and returns `Ok(user_info)`
- Maps all reqwest errors to `String` via `.map_err(|e| e.to_string())`

IMPORTANT: Use `/api/whoami-v2` NOT `/api/whoami` — the v1 endpoint returns 401 for modern fine-grained HF tokens.

**`src-tauri/src/commands/auth.rs`:**
Create four `#[tauri::command]` functions, each also annotated with `#[specta::specta]`:

1. `validate_token(token: String, state: tauri::State<'_, AppState>) -> Result<UserInfo, String>`
   - Calls `hf::client::whoami(&token).await?`
   - On success: stores token in OS keychain via `keyring::Entry::new("face-hugger", "hf-token")?.set_password(&token)?`
   - Updates in-memory `state.auth` with token and user_info
   - Returns `Ok(user_info)`

2. `get_stored_token(state: tauri::State<'_, AppState>) -> Result<Option<String>, String>`
   - Reads from in-memory state first; if token present, returns it
   - Otherwise reads from keychain via `keyring::Entry::new("face-hugger", "hf-token")?.get_password()`
   - Returns `Ok(Some(token))` or `Ok(None)` if not found
   - Catches keyring errors gracefully (NoEntry -> None, not panic)

3. `check_existing_token() -> Result<Option<String>, String>`
   - First checks keychain (same as get_stored_token keychain path)
   - Then checks `~/.cache/huggingface/token` (using `dirs::home_dir()`)
   - Then checks `~/.huggingface/token` (legacy fallback)
   - Reads file, trims whitespace, returns `Some(token)` if non-empty
   - Returns `Ok(None)` if nothing found anywhere

4. `logout(state: tauri::State<'_, AppState>) -> Result<(), String>`
   - Clears in-memory auth state (token = None, user = None)
   - Deletes keychain entry via `keyring::Entry::new("face-hugger", "hf-token")?.delete_credential()`
   - Ignores keyring deletion errors (credential may not exist)
   - Returns `Ok(())`

**`src-tauri/src/lib.rs`:**
Wire everything together with tauri-specta:
```rust
mod commands;
mod hf;
mod state;

use state::AppState;
use tauri_specta::{collect_commands, Builder};

pub fn run() {
    let builder = Builder::<tauri::Wry>::new()
        .commands(collect_commands![
            commands::auth::validate_token,
            commands::auth::logout,
            commands::auth::get_stored_token,
            commands::auth::check_existing_token,
        ]);

    #[cfg(debug_assertions)]
    builder
        .export(
            specta_typescript::Typescript::default(),
            "../src/bindings.ts",
        )
        .expect("Failed to export typescript bindings");

    tauri::Builder::default()
        .plugin(tauri_plugin_store::Builder::default().build())
        .plugin(tauri_plugin_opener::init())
        .plugin(tauri_plugin_fs::init())
        .manage(AppState::default())
        .invoke_handler(builder.invoke_handler())
        .setup(move |app| {
            builder.mount_events(app);
            Ok(())
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```
